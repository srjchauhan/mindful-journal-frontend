# Mindful Journal Frontend Service 

This React application provides a user-friendly interface for interacting with the Mindful Journal backend service, empowering users to create a personalized wellness journaling experience.

#### Check Backend service [here](https://github.com/srjchauhan/mindful-journal-backend)

# Key Features:

### 1. Seamless User Experience:
        
1. Intuitive signup and signin flows for easy access.

2. Responsive design ensures a smooth experience across different devices.
    
### 2. Interactive Journal Prompts:
        
1. Dynamically displays prompts generated by the backend's LLM, encouraging thoughtful reflection.

### 3. Journal Entry Composition:
        
1. Provides a dedicated space for users to write their journal entries.
### 4. Entry Management:
        
1. Allows users to view past entries, fostering self-awareness through revisiting experiences.

# Development Setup:
### 1. Prerequisites:

Node.js and yarn installed on your system.

### 2. Cloning the Repository:
```shell
git clone https://github.com/srjchauhan/mindful-journal-frontend.git
```

### 3. Installing Dependencies and start:

```shell
yarn install
yarn start
```


It runs the app in the development mode.
Open http://localhost:3000 to view it in the browser.
